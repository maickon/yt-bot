Como obter uma chave de API do YouTube

    Siga os passos abaixo para gerar sua chave de API e utilizá-la no seu projeto:

Passo 1: Acesse o Google Cloud Console
    
    Vá para o Google Cloud Console: https://console.cloud.google.com/
    
    Faça login com sua conta Google.

Passo 2: Crie um novo projeto
    
    No topo da página, clique no menu suspenso do seletor de projeto.
    
    Clique em "Novo Projeto".

    Dê um nome ao projeto (por exemplo: "API YouTube").
    
    Clique em "Criar".

Passo 3: Ative a API do YouTube
    
    No menu à esquerda, vá para APIs e serviços > Biblioteca.
    
    Na barra de busca, digite YouTube Data API v3.
    
    Clique na YouTube Data API v3 nos resultados.

    Clique em "Ativar".

Passo 4: Crie credenciais de API

    Após ativar a API, vá para APIs e serviços > Credenciais no menu à esquerda.

    Clique em "Criar credenciais" e selecione Chave de API.

    Sua chave de API será gerada. Copie essa chave.

Passo 5: Use sua chave de API no projeto
    
    Agora, basta inserir a chave de API no seu código Python em [youtube.py], como no exemplo abaixo:

    # Substitua pela sua chave de API do YouTube
    API_KEY = 'SUA_CHAVE_DE_API_AQUI'
